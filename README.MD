# StockFlow

Sistema moderno de controle de estoque e vendas desenvolvido em Python com interface gráfica.

---

## Funcionalidades

- Controle de estoque
- Cadastro de insumos
- Cadastro de produtos
- Receita técnica
- Venda automática
- Baixa automática de estoque
- Controle de bebidas
- Relatórios de vendas
- Interface moderna desktop
- Integração com Supabase/PostgreSQL

---

## Tecnologias utilizadas

- Python
- CustomTkinter
- Supabase
- PostgreSQL
- Git/GitHub

---

## Objetivo

Projeto desenvolvido para estudo de:
- Backend
- Banco de dados
- Sistemas empresariais
- Interfaces gráficas
- Arquitetura de software

---

## Interface

Sistema desktop com:
- Sidebar moderna
- Dashboard
